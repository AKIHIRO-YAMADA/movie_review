# 映画レビューサイト

## サイト概要
ユーザーがお勧めする映画を投稿し、コメントや評価ができるサイト。

### サイトテーマ
流行の映画だけでなく、名作に出会える。


### テーマを選んだ理由
外出自粛が続いており家で時間を過ごすことが増えたので、そういった方にサイトを使ってもらい、質の良い映画を見てほしいと思ったから。

### ターゲットユーザ
外出を自粛してる人
良い作品を選びたい人

### 主な利用シーン
家で映画を見る時。
